# XP Roadmap

This document describes the Enonic XP roadmap. Items can be linked to issues or feature documents. Items can also be annotated with the following icons:

* :moneybag: - means enterprise feature.
* :rocket: - promised to customers.

* Add Caption, Artist, Copyright to Vector images in Media
