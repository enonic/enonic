# CMS Expert role

As a content studio user, I should not be able to see the button "Source code" of HTML areas and Text areas if I do not have one of the following roles:  “cms.expert”, “cms.admin” or “system.admin”.

## Estimate

1 day

## Requirements

* In Content Studio, the button "Source code" of HTML areas and Text areas is hidden if the user does not have one of the following roles:  “cms.expert” or “cms.admin” “system.admin”.
* A role "cms.expert" is created on system repository initialization. Similar to "cms.cms.app", the role has read rights on root content by default.

## Acceptance Criteria

* All requirements are met
