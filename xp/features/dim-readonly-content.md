# Dim content where the user has only read access (Content Studio)

As a user, I want content items that I have only read-only access to to be visually highlighted in the Content Grid.

## Estimate

2 hours

## Requirements

* A content item with read-only access must have a "dimmed" effect in the Content Grid
* A tooltip that says "Read-only" must be shown on hover of a content item with read-only access

## Dependencies

None

## Acceptance Criteria

* All requirements are met
* User test is satisfactory
